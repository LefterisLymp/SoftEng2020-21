Install node.js από το web

Στο cmd σαν administrator:
	npx create-react-app frontend

Για να γίνει install το bootstrap:
	npm install react-bootstrap bootstrap
	npm install jquery
	npm install popper.js
	npm install react-rooter-dom
	npm install react-chartjs-2
	npm install --save ng2-charts
	npm install --save chart.js
	npm install react-dropdown  --save

Στο cmd πάλι
	cd .../frontend
	npm start

Και η εφαρμογή τρέχει στη θύρα localhost:3000
